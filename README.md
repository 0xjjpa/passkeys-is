# Passkeys.is

Your source for everything Passkeys from a developer perspective, focused in documenting particular questions about the usage, support, and edge-cases around them.

# Setup project

```bash
npm install
```

# Run dev 

```bash
npm run dev
```

# Build + run prod

```bash
npm run build && npm run start
```
